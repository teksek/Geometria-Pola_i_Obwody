# Kalkulator Geometryczny

Konsolowa aplikacja w C# do obliczania pól i obwodów figur geometrycznych.

## Funkcjonalności

Obliczanie podstawowych parametrów:
- Pola figur płaskich (kwadrat, prostokąt, trójkąt, trapez, romb, etc.)
- Obwody figur
- Wysokości
- Przekątne
- Pola całkowite brył
- Objętości

## Technologie

C# (.NET Console Application)

## Uruchomienie
```bash
git clone https://github.com/teksek/Geometria-Pola_i_Obwody.git
cd Geometria-Pola_i_Obwody
dotnet run
```
